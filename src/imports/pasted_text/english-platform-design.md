Создай полноценный, современный, эстетичный дизайн веб-платформы по английскому языку. Это не лендинг и не обычная библиотека файлов, а умная образовательная платформа, где удобно искать, просматривать и использовать материалы для учебы, подготовки к олимпиадам, ЕГЭ и урокам. Дизайн должен быть визуально целостным, премиальным, аккуратным, дружелюбным и сразу считываться как сайт именно по английскому языку.

Ключевая идея интерфейса:
платформа должна ощущаться как digital language lab / english study workspace / academic library нового поколения. Не делай банальный набор одинаковых карточек. Нужен характерный визуальный язык, связанный с английским: словари, языковые заметки, speaking cards, grammar blocks, notebook tabs, editorial academic feeling, мягкая “британская” эстетика без прямой перегрузки флагами и клише.

Визуальный стиль:
— современный академический минимализм
— ощущение интеллектуального, структурного, языкового пространства
— сочетание цифрового интерфейса и редакционно-учебной эстетики
— много воздуха, аккуратная типографика, ясная иерархия
— интерфейс не “детский”, а умный, чистый, стильный и удобный для старшеклассников, олимпиадников, преподавателей и методистов
— должны быть визуальные ассоциации с английским языком: subtle quote marks, phonetic accents, bookmarks, structured tabs, language chips, speech cards, word blocks, academic dividers, tone of notebooks/dictionaries/editorial layouts

Цветовая концепция:
сделай приятную, узнаваемую и спокойную палитру, которая ассоциируется с английским языком и academic English-средой.
Используй:
— глубокий тёмно-синий как базовый акцент: #1E2A44
— мягкий холодный красно-бордовый как второй акцент: #A9445B
— тёплый молочный / paper white для фона: #F7F4EE
— светлый серо-голубой для вторичных поверхностей: #DCE6F2
— приглушённый серо-графитовый для текста: #2B2F36
— светлый белый для карточек: #FFFFFF
— мягкий шалфейный или muted green для статусов успеха и “готово”: #A8B8A1
— деликатный золотисто-бежевый для маленьких акцентных меток: #D8C3A5

Важно:
— не делай яркий кислотный дизайн
— не делай тяжелый корпоративный интерфейс
— не делай слишком школьный стиль
— не используй прямую перегрузку британским флагом
— можно очень деликатно использовать логику британской палитры: navy + burgundy + warm paper + mist blue
— интерфейс должен быть приятным и узнаваемым как сайт по английскому языку уже за счет палитры, типографики, элементов и подачи контента

Типографика:
— современный чистый sans-serif для интерфейса: Inter или Manrope
— для крупных заголовков или разделительных акцентов можно добавить элегантный serif-акцент: Lora или Playfair Display
— основной текст должен быть очень читабельным
— крупная ясная иерархия
— заголовки уверенные, но не тяжелые
— использовать аккуратные капсульные метки, language tags, numbered grammar blocks, section labels
— можно локально использовать английские микроэлементы в интерфейсе: “Grammar”, “Speaking”, “Writing”, “Use of English”, “Workbook”, “Notes”, “Preview”, но общий интерфейс русскоязычный

Создай дизайн-систему:
1. Цветовые токены
2. Типографическая шкала
3. Сетка
4. Компоненты
5. Состояния hover / active / selected / disabled
6. Стили тегов, фильтров, кнопок, карточек, таблиц, полей поиска, модальных окон, dropdown, загрузки файлов
7. Единые отступы, радиусы, тени, бордеры

Общее UI-направление:
— desktop-first
— адаптивность для tablet
— чистая сетка
— мягкие большие радиусы 18–24 px
— тонкие бордеры
— легкие тени
— аккуратные вкладки
— липкая навигация
— плавные hover-состояния
— ощущение собранного, дорогого, умного продукта

Структура сайта:

1. Главная страница
Нужен сильный first screen.
Сверху:
— логотип или wordmark
— меню
— кнопка входа в админ-панель в правом верхнем углу
Hero-блок:
— крупный заголовок
— короткое позиционирование платформы
— большой умный поиск по материалам
— быстрые сценарии входа:
   Учебники
   Олимпиады
   ЕГЭ
   Подготовка к урокам
Ниже:
— блок “Выберите цель”
— блок “Популярные темы”
— блок “Новые материалы”
— блок “Подборки”
— блок с визуальным превью killer feature
— аккуратный footer

Главная должна выглядеть как продуманный образовательный продукт, а не как шаблонный landing page.

2. Страница-каталог / workspace layout
Сделай основной режим платформы в формате 3-колоночного умного интерфейса:
— слева вертикальная навигация по разделам и подразделам
— в центре список материалов
— справа панель предпросмотра / информации / быстрых действий
Такой layout должен стать главной фишкой интерфейса.

Левая колонка:
— разделы:
   Учебники
   Подготовка к олимпиадам
   Подготовка к ЕГЭ
   Подготовка к урокам
— раскрывающиеся подразделы

Подразделы:
Учебники:
— Грифованные РФ

Подготовка к олимпиадам:
— Grammar
— Use of English
— Cultural Awareness
— Writing
— Speaking

Подготовка к ЕГЭ:
— Grammar
— Writing
— Speaking

Подготовка к урокам:
— Раздаточный материал
— Методические разработки уроков по методике CLIL
— Методические разработки уроков по SELTA

Центральная колонка:
— заголовок раздела
— подзаголовок / описание
— строка поиска
— фильтры
— сортировка
— режим отображения: cards / compact list
— список материалов

Правая колонка:
— превью выбранного материала
— метаданные
— теги
— связанные материалы
— кнопки “Открыть”, “Скачать”, “Скопировать ссылку”, “В избранное”

3. Раздел “Учебники”
Дизайн должен показывать, что это не просто список файлов, а организованная библиотека.
Карточка учебника:
— обложка
— название
— автор
— издательство
— класс / уровень
— краткое описание
— attached items: textbook / workbook / audio / teacher’s book / extra materials
— кнопки действий
Добавь аккуратную навигацию по уровням, классам и авторам.

4. Раздел “Подготовка к олимпиадам”
Сделай более интеллектуальный и “олимпиадный” стиль внутри этого раздела:
— визуальные маркеры сложности
— badges по уровням
— tags по типу заданий
— быстрые подборки
Подразделы должны выглядеть по-разному на уровне микроакцентов, но оставаться в одной системе:
Grammar — ощущение структурности
Use of English — compact task bank
Cultural Awareness — более editorial feel
Writing — blocks with criteria and templates
Speaking — cards and speech prompts

5. Раздел “Подготовка к ЕГЭ”
Этот раздел должен быть максимально прикладным и чуть более строгим.
Покажи:
— готовые шаблоны
— тренировочные задания
— критерии оценивания
— speaking cards
— тайминг
— чек-листы
Важно визуально отделить ЕГЭ-раздел от олимпиадного:
олимпиадный — умнее и глубже
ЕГЭ — практичнее, четче, экзаменационнее

6. Раздел “Подготовка к урокам”
Нужен более профессиональный teacher workspace feel.
Покажи:
— worksheets
— lesson plans
— CLIL materials
— SELTA materials
— фильтры по педагогической задаче:
   ввести лексику
   закрепить грамматику
   развить speaking
   подготовить writing
   провести интегрированный урок
   взять готовый урок быстро
Здесь можно добавить более “методический” ритм интерфейса: структурные карточки, lesson templates, print-ready marks.

7. Страница карточки материала
Сделай очень качественную detail page.
Структура:
— название
— breadcrumb
— тип материала
— уровень
— раздел / подраздел
— автор / редактор
— дата обновления
— описание
— attached files
— preview area
— related materials
— recommendations
— теги
— блок “что внутри”
Карточка должна ощущаться как страница качественного образовательного ресурса.

8. Killer feature: Skill Map Navigator
Создай отдельный экран или крупный блок, показывающий ключевую фишку платформы.
Суть:
материалы связаны с языковыми микроскиллами.
Примеры микроскиллов:
— Tenses
— Passive Voice
— Conditionals
— Word Formation
— Linking Devices
— Essay Structure
— Argumentation
— Cultural Context
— Speaking Fluency
— Pronunciation Patterns

Визуально покажи это как интерактивную карту навыков:
— skill nodes
— связи между навыками
— прогресс
— рекомендованные материалы
— путь изучения
— фильтр по целям: Олимпиада / ЕГЭ / Урок / Общая подготовка
Стиль должен быть визуально сильным и современным, но не перегруженным.
Это должен быть один из самых эффектных экранов всей платформы.

9. Админ-панель
Создай отдельный полноценный дизайн редакторской админ-панели.
Не просто таблицу, а удобный content management workspace.

Важные UX-принципы:
— пользователь должен за несколько секунд понимать, где он находится
— поиск и фильтры всегда под рукой
— материал можно быстро открыть без лишних переходов
— интерфейс должен выглядеть интеллектуально, но не сложно
— навигация должна быть очень ясной
— акцент на удобство реального использования
— интерфейс должен быть эстетичным в статике и логичным в работе

Микродетали, которые усилят ассоциацию с английским:
— аккуратные speech-bubble элементы
— quote marks как декоративные акценты
— phonetic / language-lab стилистика в иконках
— dictionary-tab layout
— vocabulary chips
— notebook-like section dividers
— карточки с языковыми категориями
— subtle underline / mark-up graphics, как в хороших учебных заметках
— можно использовать маленькие графические элементы, напоминающие про speaking, writing, reading, grammar, но очень деликатно

Важно не делать:
— шаблонный EdTech с кислотными цветами
— обычный marketplace
— скучный корпоративный кабинет
— школьный мультяшный стиль
— прямую копию библиотечного сайта
— перегруженные иллюстрации
— слишком много красного, синего и белого в лоб
— хаотичные карточки без системы

Нужно получить:
— несколько полноценных экранов сайта
— единый дизайн-язык
— сильную главную
— мощный основной workspace-экран
— detail page материала
— экран skill map
— вход в админку
— dashboard админки

— набор основных компонентов

Результат должен выглядеть как сильный, современный, визуально узнаваемый, эстетичный и практичный веб-продукт по английскому языку.
